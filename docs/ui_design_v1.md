# 印度端子工廠：整合門戶 UI/UX 設計規範 (V1.0)

## 1. 視覺風格 (Visual Identity)
本系統採購 **「工業 4.0」** 現代美學，旨在展現端子工廠的專業感與精確性。

- **配色方案:**
    - 主色: `#1E3A8A` (深海藍 - 代表穩定)
    - 輔色: `#64748B` (板岩灰 - 代表工業)
    - 背景: `#F8FAFC` (亮白灰 - 清晰閱讀) / 暗色模式支援。
- **字體:** 使用 Google Fonts - **Inter** 或 **Outfit**，確保長篇數據的可讀性。

## 2. 核心介面配置 (Layout)

### 2.1 統一登入頁 (Master Login)
- **設計:** 中央玻璃擬態登入框。
- **邏輯:** 驗證 Nexus 帳號中心，根據 `module_permissions` 決定登入後可見的模組。

### 2.2 全域儀表板 (Global Dashboard)
- **頂部看板:** 
    - 印度工廠今日概況 (今日採購總額、倉庫庫存異動、零用金報帳數)。
- **系統導航卡片:** 
    - 零用金系統卡 (Petty Cash)。
    - 採購管理系統卡 (Procurement)。
    - 印度倉庫 WMS 卡 (WMS)。

### 2.3 導航欄 (Sidebar/Navbar)
- **多系統選單:** 使用者無需跳轉分頁，點選後在主區域直接切換內容（透過 React Router）。

## 3. 使用者交互 (Interactions)
- **微動畫:** 按鈕點擊的回彈效果、數據載入時的骨架屏 (Skeleton Screens)。
- **即時回饋:** 所有的跨系統同步動作（如 PO 轉入庫）都會有明確的成功動畫提示。
